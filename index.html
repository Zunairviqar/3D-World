Instructions

- Use the W,S,A,D keys to move around the world.

Documentation

The aim of this project is to produce a 3D environment, which can be explored through the use of W, A, S & D and the mouse. This assignment is currently only the first version of our final project where this project will be further developed. Additional developments will allow the user to visualize themselves in front of the screen as a car which they can drive around to explore the world. Physics including collisions and gravity will also be included to give this project a more real world feel.

There will be some more potential developments to the project which are still in discussion such as a maze that would have to be completed in 3D using the car. Additionally, we plan on using the Web RTC Library that we used for Assignment 06, in order to potentially allow multiple cars to coexist in a specific game and roam around, or possibly, if the skills allow, race with one another. As we progress with the project and get to know of the capabilities of the A-Frame Library, we can develop and polish our idea further!

Technologies Used

We will be working with the VR framework (A-frame) where a special technique will need to be used to position the camera in such a way that it is facing downwards onto our user and from a specific distance at a certain angle at all times. We will also be implementing physics and mechanics to our user objects as well as all the other objects in the game so we will be taking help from: https://github.com/n5ro/aframe-physics-system. We will also need to use our prior knowledge of p5 regarding classes and objects and user movements wherever needed.
